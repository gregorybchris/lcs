# Longest Common Subsequence

Swift implementations of longest common subsequence (LCS) and longest increasing subsequence (LIS)

| Algorithm  | Time     | Space          |
| ---------- | -------- | -------------- |
| LCS        | `O(mn)`  | `O(mn)`        |
| LCS length | `O(mn)`  | `O(min{m, n})` |
| LIS        | `O(n^2)` | `O(n^2)`       |
